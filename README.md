# 🍗 Recipe Card Page – Classic Herb Roasted Chicken

A visually appealing recipe details page built using HTML and CSS.  
This project demonstrates layout structuring, component styling, and clean UI design for a food recipe interface.

---

## 📌 Project Overview

This project displays a detailed recipe card for **Classic Herb Roasted Chicken**, including:

- Hero section with recipe image and title  
- Preparation time, cook time, and servings  
- Ingredients checklist  
- Step-by-step cooking instructions  
- Nutritional information  
- Print button UI  

The goal of this project is to practice modern layout techniques and build a structured content page with clear visual hierarchy.

---

## 🛠️ Built With

- HTML5  
- CSS3  
- Flexbox
